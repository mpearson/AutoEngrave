# AutoEngrave
Web-based laser cutting and engraving platform.

# Requirements:

## Back end
- numpy
- opencv-python
- cairosvg
- flask
- SQLite

## Front end
- nodejs
- react/redux
- typescript

# Features
- Design templates
- Design library
- Multi-task engravings
- G-code export
- serial port scanning/selection

# Roadmap
- G-code sending
- job pause/resume/cancel
- job queuing
- template loading
- cutting/vectoring
- pre/post g-code

- tag search
- bulk tag search (e.g. paste comma-separated tags)
- make ignoring outlines optional
